# AGENTS.md

## OpenSpec 文档语言

- 在创建或修改 `openspec/changes/` 下的文档时，正文默认使用中文，方便审核。
- `spec.md` 中保留 OpenSpec 需要解析的结构关键字，例如 `## ADDED Requirements`、`### Requirement:`、`#### Scenario:`、`WHEN`、`THEN`。
- `tasks.md` 中保留 checkbox 格式，例如 `- [ ] 1.1 ...` 或 `- [x] 1.1 ...`，任务描述可以写中文。
- `proposal.md` 和 `design.md` 可以使用中文正文；如需兼容工具识别，优先保留英文主标题，并在标题后补充中文说明。
